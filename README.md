# Genie: A Chess AI  

Genie is a cutting-edge Chess AI designed to predict optimal moves dynamically, inspired by the AlphaZero learning methodology. Genie evaluates possible move sequences and adapts its strategies in real-time to enhance gameplay.  

## Features  
- Predicts the top three possible moves for both players.  
- Dynamically adapts decision-making based on board changes.  
- Evaluates multiple move sequences to select the best possible move.  

## Folder Structure  
- `data/`: Contains chess datasets and training data.  
- `src/`: The source code for Genie, including the AI and chess engine logic.  
- `notebooks/`: Jupyter notebooks for testing and experimenting with models.  

## Installation  

1. Clone the repository:  
   ```bash  
   git clone https://github.com/yourusername/genie.git  
   cd genie  
   ```  

2. Install dependencies:  
   ```bash  
   pip install -r requirements.txt  
   ```  

## Usage  

1. Add your chess datasets to the `data/` folder.  
2. Run the AI scripts from the `src/` folder to start training or evaluation.
